## 表达式

> 表达式是运算符和操作符所构成的序列

### 表达式的顺序
| 运算符                 | 描述                                                 |
| :--------------------- | :--------------------------------------------------- |
| **                     | 指数（最高优先级）                                   |
| \*、\/、%、\/\/        | 按位翻转，一元加号和减号（最后两个的方法名为+@和-@） |
| +、-                   | 加法、减法                                           |
| >>、<<                 | 右移、左移运算符                                     |
| &                      | 位与                                                 |
| ^、\|                  | 位运算符                                             |
| <=、<、>、>=、<> == != | 比较运算符                                           |
| =%=\/=\/=-=+=*=\**=       | 赋值运算符                                           |
| is 、is not            | 身份运算符                                           |
| in、in not             | 成员运算符                                           |
| not、or and            | 逻辑运算符(and优先级高于)                                           |

1.  先乘除后加减
2.  幂运算优先级最高
3.  三元运算符优先级最低
4.  元数越少，优先级越高  
    幂运算：一元运算符
5.  优先级从高到低排列：  
    幂运算  
    正负号  
    算数运算符（— = * /）  
    比较运算符（< > <= >=）  
    逻辑运算符（not and or）
