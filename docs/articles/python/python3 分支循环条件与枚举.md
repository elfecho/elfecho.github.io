## 表达式

> 表达式是运算符和操作符所构成的序列

### 表达式的顺序

| 序号               | 运算符                 | 描述                                                 |
| :--------------------- | :--------------------------------------------------- | :--------------------------------------------------- |
| 1                   | **                     | 指数（最高优先级）                                   |
| 2       | \*、\/、%、\/\/        | 按位翻转，一元加号和减号（最后两个的方法名为+@和-@） |
| 3                  | +、-                   | 加法、减法                                           |
| 4                | >>、<<                 | 右移、左移运算符                                     |
| 5                     | &                      | 位与                                                 |
| 6                 | ^、\|                  | 位运算符                                             |
| 7 | <=、<、>、>=、<> == != | 比较运算符                                           |
| 8      | =%=\/=\/=-=+=*=\**=       | 赋值运算符                                           |
| 9           | is 、is not            | 身份运算符                                           |
| 10           | in、in not             | 成员运算符                                           |
| 11          | not、or and            | 逻辑运算符(not -> and -> or 优先级从高到底)                        |

1.  先乘除后加减
2.  幂运算优先级最高
3.  三元运算符优先级最低
4.  元数越少，优先级越高  
    幂运算：一元运算符
5.  优先级从高到低排列：  
    幂运算  
    正负号  
    算数运算符（— = * /）  
    比较运算符（< > <= >=）  
    逻辑运算符（not and or）

示例：
```python
a=1
b=2
c=3
not a or b + 2 == c #这个算式如何使用括号将优先级区分开呢
```

首先观察有`not`跟`or`的逻辑运算符、`+`的算术运算符、`==`的比较运算符

然后进行比较，算术运算符是最高的，然后到比较运算符、最后就是逻辑运算符not -> or了

```
(not a) or ((b + 2) == c) # False
```
